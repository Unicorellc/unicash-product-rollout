# UniCash Product Rollout Plan

An interactive presentation showcasing UniCash's comprehensive product rollout strategy, including:

- **5 Product Lines**: Cash advances, remittances, assistance plans, auto insurance, and burial assistance
- **12-Month Timeline**: Phased rollout from founding team to 250K users
- **Market Opportunity**: $167B addressable market serving 62M underbanked Americans
- **Investment Details**: $4M raise with detailed budget breakdown

## View the Presentation

Open `index.html` in your browser to view the interactive scrollable presentation.

## Features

- Smooth scroll navigation
- Animated statistics and counters
- Interactive expandable cards
- Mobile-responsive design
- Professional gradient design system

## Technology Stack

Pure HTML/CSS/JavaScript - no dependencies required.
